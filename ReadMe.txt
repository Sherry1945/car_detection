运行说明： 环境需要 python>=3.6.6 openCV>=3.4.3
文件夹中包含代码以及一个监控视频示例以及readme
配置好环境后，需将最后一行 监控视频路径 改为 配套监控视频在本地的路径然后运行即可
结果会实时显示在监控视频左上角，以及 terminal 中最后一行会显示探测到的车辆总数
The environment requires Python >=3.6.6 openCV>=3.4.3  
The folder contains the code as well as a sample surveillance video and readme  
After the environment is configured, change the last line of the monitoring video path to the local path of the supporting monitoring video and run it  
The results will be displayed in the upper left corner of the surveillance video in real time, and the total number of detected vehicles will be displayed in the last line of terminal
